# UBCustomPaletteLoader
Custom palette loader for UNBEATABLE 
Not affiliated or endorsed by D-CELL Games

Requires [MelonLoader](https://github.com/LavaGang/MelonLoader/releases/latest)

Edit the TXT file to change the colors then select the "custom" palette in game like you would any other palette

Really just use [this one](https://github.com/Zachava96/CustomPalettes) instead.
I made this without knowing this other one existed, It's better than this one.
